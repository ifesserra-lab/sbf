## US 2.1: Criar Repositório E Projeto No GitHub

Como equipe do projeto,
quero ter um repositório e um GitHub Project no `ifesserra-lab`,
para gerenciar código, documentação, tarefas e evolução da landing page.

Contexto:
O GitHub será usado para versionar o trabalho e dar visibilidade ao andamento.

Critérios de aceitação:
- Dado que o projeto está em andamento, quando a equipe acessa o GitHub, então encontra o repositório oficial.
- Dado que o backlog está definido, quando a equipe acessa o GitHub Project, então encontra os itens principais do projeto.
- Dado que uma tarefa muda de estado, quando o projeto é atualizado, então o status reflete a situação real.

Prioridade:
Alta

Estimativa:
M

Dependências:
- Acesso administrativo à organização `ifesserra-lab`.
- Definição do nome oficial do repositório/projeto.

Status:
A fazer

| Task | Resultado esperado | Status |
